<h1 align="center">💸 Refund</h1>

<p align="center">  
Uma aplicação web leve e intuitiva para gerenciar despesas pessoais de forma prática e organizada.  
</p>

---

## 🧭 Funcionalidades

- Adicionar novas despesas com título, valor e data  
- Listar todas as despesas registradas  
- Remover despesas individualmente  
- Exibir o total de despesas atualizado  
- Interface leve, responsiva e fácil de usar

## 🎯 Objetivo

O Refund foi criado como um projeto de prática para reforçar conhecimentos em:

- Manipulação do DOM com JavaScript

- Uso de LocalStorage para persistência de dados

- Estruturação limpa de projetos front-end

- Desenvolvimento de interfaces simples e funcionais
